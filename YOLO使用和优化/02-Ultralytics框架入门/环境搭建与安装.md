# 环境搭建与安装

> **目标**: 配置完整的YOLO开发环境，包括Python、PyTorch、CUDA和Ultralytics

---

## 📋 系统要求

### 最低配置

| 组件 | 最低要求 | 推荐配置 |
|------|----------|----------|
| **操作系统** | Windows 10/11, Ubuntu 18.04+, macOS 10.15+ | Windows 11, Ubuntu 22.04 |
| **Python** | 3.8+ | 3.9 - 3.11 |
| **内存** | 8 GB RAM | 16 GB RAM |
| **GPU** | 不需要（CPU模式） | NVIDIA GPU（4GB+ VRAM） |
| **存储空间** | 5 GB 可用空间 | 20 GB SSD |

### 推荐硬件（用于训练）

- **GPU**: NVIDIA RTX 3060 (12GB) 或更高
- **显存**: 至少 8GB（推荐 12GB+ 用于训练大模型）
- **CUDA**: 11.8+ / 12.x
- **cuDNN**: 8.6+

## 🚀 快速安装指南

### 方法一：pip 安装（推荐）

```bash
# 1. 创建虚拟环境（强烈推荐）
python -m venv yolo_env

# 激活虚拟环境
# Windows:
yolo_env\Scripts\activate
# Linux/macOS:
source yolo_env/bin/activate

# 2. 升级 pip
pip install --upgrade pip

# 3. 安装 ultralytics（会自动安装 PyTorch 等依赖）
pip install ultralytics

# 4. 验证安装成功
yolo version
```

**预期输出**:

```

Setup complete ✅ (8 CPUs, 31.7 GB RAM, 781.4/1000 GB disk)
```

## 🔧 GPU 环境配置（重要！）

### 检查 CUDA 环境

```python
import torch

def check_gpu_environment():
    """检查 GPU 和 CUDA 环境"""
    print("=" * 60)
    print("🔍 GPU 环境检测报告")
    print("=" * 60)
    
    # PyTorch 版本
    print(f"\n✅ PyTorch 版本: {torch.__version__}")
    
    # CUDA 是否可用
    cuda_available = torch.cuda.is_available()
    print(f"🎮 CUDA 可用: {'是 ✅' if cuda_available else '否 ❌'}")
    
    if cuda_available:
        # CUDA 版本
        print(f"📦 CUDA 版本: {torch.version.cuda}")
        
        # cuDNN 版本
        print(f"⚡ cuDNN 版本: {torch.backends.cudnn.version()}")
        
        # GPU 数量
        num_gpus = torch.cuda.device_count()
        print(f"🖥️  GPU 数量: {num_gpus}")
        
        for i in range(num_gpus):
            gpu = torch.cuda.get_device_name(i)
            gpu_cap = torch.cuda.get_device_capability(i)
            gpu_mem = torch.cuda.get_device_properties(i).total_mem / 1024**3
            
            print(f"\n   GPU {i}: {gpu}")
            print(f"      计算能力: {gpu_cap[0]}.{gpu_cap[1]}")
            print(f"      显存大小: {gpu_mem:.1f} GB")
            
        # 当前使用的 GPU
        current_device = torch.cuda.current_device()
        print(f"\n💻 当前使用 GPU: {current_device} ({torch.cuda.get_device_name(current_device)})")
    
    else:
        print("\n⚠️  未检测到 GPU，将使用 CPU 模式运行")
        print("   如需 GPU 加速，请:")
        print("   1. 安装支持 CUDA 的 PyTorch 版本")
        print("   2. 安装正确的 NVIDIA 驱动程序")
    
    print("\n" + "=" * 60)


if __name__ == '__main__':
    check_gpu_environment()
```

## 🐳 Docker 容器化部署（生产环境推荐）

### Dockerfile 示例

```dockerfile
# 基础镜像：包含 CUDA 的 PyTorch 官方镜像
FROM pytorch/pytorch:2.1.0-cuda12.1-cudnn8-runtime-ubuntu22.04

# 设置工作目录
WORKDIR /app

# 安装系统依赖
RUN apt-get update && apt-get install -y \
    git \
    ffmpeg \
    libsm6 \
    libxext6 \
    && rm -rf /var/lib/apt/lists/*

# 升级 pip 并安装 ultralytics
RUN pip install --no-cache-dir --upgrade pip && \
    pip install --no-cache-dir ultralytics opencv-python-headless

# 复制项目文件
COPY . .

# 默认命令
CMD ["python", "main.py"]
```

### docker-compose.yml

```yaml
version: '3.8'

services:
  yolo-training:
    build: .
    container_name: yolo-trainer
    volumes:
      - ./data:/app/data          # 数据集目录
      - ./models:/app/models      # 模型保存目录
      - ./runs:/app/runs          # 训练日志目录
    environment:
      - CUDA_VISIBLE_DEVICES=0     # 使用第0号GPU
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
    command: python train.py

  yolo-inference:
    build: .
    container_name: yolo-inference
    ports:
      - "8000:8000"
    volumes:
      - ./models:/app/models
    environment:
      - MODEL_PATH=models/best.pt
    command: uvicorn api:app --host 0.0.0.0 --port 8000
```

### 构建和运行

```bash
# 构建镜像
docker-compose build

# 启动训练容器
docker-compose up yolo-training

# 启动推理服务
docker-compose up yolo-inference

# 进入容器调试
docker exec -it yolo-trainer bash
```

### 完整测试脚本

```python
import sys
import torch
from pathlib import Path


def test_ultralytics_installation():
    """全面测试 ultralytics 安装"""
    
    tests_passed = []
    tests_failed = []
    
    print("\n" + "=" * 70)
    print("🧪 Ultralytics 安装验证测试")
    print("=" * 70)
    
    # 测试 1: 导入模块
    try:
        from ultralytics import YOLO
        tests_passed.append("✅ ultralytics 导入成功")
    except ImportError as e:
        tests_failed.append(f"❌ 导入失败: {e}")
        return False
    
    # 测试 2: 检查版本
    import ultralytics
    version = ultralytics.__version__
    tests_passed.append(f"✅ Ultralytics 版本: {version}")
    
    # 测试 3: PyTorch 和 CUDA
    tests_passed.append(f"✅ PyTorch 版本: {torch.__version__}")
    
    if torch.cuda.is_available():
        tests_passed.append(f"✅ CUDA 版本: {torch.version.cuda}")
        tests_passed.append(f"✅ GPU 设备: {torch.cuda.get_device_name(0)}")
    else:
        tests_failed.append("⚠️  CUDA 不可用，将使用 CPU 模式")
    
    # 测试 4: 加载预训练模型
    try:
        model = YOLO('yolov8n.pt')
        tests_passed.append("✅ 预训练模型加载成功 (yolov8n.pt)")
    except Exception as e:
        tests_failed.append(f"❌ 模型加载失败: {e}")
        return False
    
    # 测试 5: 简单推理测试
    try:
        import numpy as np
        # 创建随机图像进行测试
        test_image = np.random.randint(0, 255, (640, 640, 3), dtype=np.uint8)
        results = model(test_image, verbose=False)
        if len(results) > 0:
            tests_passed.append("✅ 推理测试通过")
        else:
            tests_failed.append("⚠️  推理返回空结果")
    except Exception as e:
        tests_failed.append(f"❌ 推理失败: {e}")
    
    # 测试 6: 检查依赖包
    required_packages = {
        'opencv-python': 'cv2',
        'matplotlib': 'matplotlib',
        'pandas': 'pandas',
        'Pillow': 'PIL',
        'scipy': 'scipy',
        'requests': 'requests',
        'tqdm': 'tqdm',
        'psutil': 'psutil'
    }
    
    for pkg, module in required_packages.items():
        try:
            __import__(module)
            tests_passed.append(f"✅ {pkg} 已安装")
        except ImportError:
            tests_failed.append(f"⚠️  {pkg} 未安装 (可选)")
    
    # 输出结果
    print("\n📊 测试结果汇总:\n")
    
    for result in tests_passed:
        print(result)
    
    for result in tests_failed:
        print(result)
    
    print("\n" + "-" * 70)
    total_passed = len(tests_passed)
    total_tests = len(tests_passed) + len(tests_failed)
    success_rate = (total_passed / total_tests) * 100
    
    print(f"✅ 通过: {total_passed}/{total_tests} ({success_rate:.1f}%)")
    
    if success_rate >= 80:
        print("🎉 安装验证完成，环境配置正确！")
        return True
    else:
        print("⚠️  部分测试未通过，请检查上述错误信息。")
        return False


def check_disk_space():
    """检查磁盘空间"""
    disk_usage = psutil.disk_usage('/')
    free_gb = disk_usage.free / (1024**3)
    
    print("\n💾 磁盘空间检查:")
    print(f"   总空间: {disk_usage.total / (1024**3):.1f} GB")
    print(f"   已使用: {disk_usage.used / (1024**3):.1f} GB")
    print(f"   可用空间: {free_gb:.1f} GB")
    
    if free_gb < 5:
        print("   ⚠️  磁盘空间不足，建议清理后继续")
        return False
    else:
        return True


if __name__ == '__main__':
```

## 🔍 常见安装问题及解决方案

### 问题 1: CUDA out of memory

**原因**: GPU显存不足或CUDA版本不匹配

**解决方案**:

```python
# 方案 A: 减小 batch size
model.train(data='data.yaml', batch=8)  # 从默认16减小到8

# 方案 B: 使用混合精度训练
model.train(
    data='data.yaml',
    amp=True,           # 自动混合精度，节省约50%显存
    batch=16
)

# 方案 C: 使用更小的模型
model = YOLO('yolov8n.pt')  # 使用 nano 版本而非 large/xlarge
```

---

### 问题 2: ModuleNotFoundError: No module named 'ultralytics'

**解决方案**:

```bash
# 确认虚拟环境已激活
# Windows:
where python
# 应该显示: your_path\ylo_env\Scripts\python.exe

# Linux/macOS:
which python
# 应该显示: your_path/ylo_env/bin/python

# 重新安装
pip uninstall ultralytics -y
pip install ultralytics --no-cache-dir

# 验证安装位置
pip show ultralytics
```

---

### 问题 3: OpenCV 相关错误

```bash
# 安装 headless 版本（无GUI依赖，适合服务器）
pip uninstall opencv-python -y
pip install opencv-python-headless

# 或者安装完整版（需要本地GUI）
pip install opencv-python
```

### 问题 4: 权限问题（Windows）

```bash
# 以管理员身份运行 PowerShell 或 CMD
# 或者在用户目录下创建虚拟环境

%USERPROFILE%\ylo_env\Scripts\activate
```

### 一键安装脚本

**install_env.sh** (Linux/macOS):

```bash
#!/bin/bash

set -e  # 遇到错误立即退出

echo "========================================"
echo "🚀 YOLO 开发环境一键安装脚本"
echo "========================================"

# 检查 Python 版本
PYTHON_VERSION=$(python3 --version 2>&1 | awk '{print $2}')
echo "✅ Python 版本: $PYTHON_VERSION"

# 创建虚拟环境
echo ""
echo "📦 创建虚拟环境..."
python3 -m venv ylo_env
source ylo_env/bin/activate

# 升级 pip
echo ""
echo "⬆️  升级 pip..."
pip install --upgrade pip

# 选择 CUDA 版本
echo ""
echo "请选择 PyTorch CUDA 版本:"
echo "1) CUDA 11.8 (推荐 RTX 30 系列)"
echo "2) CUDA 12.1 (推荐 RTX 40 系列)"
echo "3) CPU only"
read -p "输入选项 [1/2/3]: " choice

case $choice in
    1)
        echo "🔥 安装 PyTorch (CUDA 11.8)..."
        pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
        ;;
    2)
        echo "🔥 安装 PyTorch (CUDA 12.1)..."
        pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
        ;;
    3)
        echo "🔥 安装 PyTorch (CPU only)..."
        pip install torch torchvision torchaudio
        ;;
    *)
        echo "❌ 无效选项，默认安装 CPU 版本"
        pip install torch torchvision torchaudio
        ;;
esac

# 安装 ultralytics 及其他依赖
echo ""
echo "📦 安装 ultralytics 和其他依赖..."
pip install -r requirements.txt

# 验证安装
echo ""
echo "🧪 验证安装..."
python test_installation.py

echo ""
echo "========================================"
echo "✅ 安装完成！"
echo ""
echo "激活环境:"
echo "  source ylo_env/bin/activate"
echo ""
echo "快速测试:"
echo "  yolo predict model=yolov8n.pt source=test.jpg"
echo "========================================"
```
